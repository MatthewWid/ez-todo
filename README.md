## Development

Install Node:

```bash
curl -L https://git.io/n-install | bash
n auto
```

Install pnpm:

```bash
npm i -g pnpm
```

Install dependencies:

```bash
pnpm i
```

Run development server:

```bash
pnpm run dev
```

Build for production:

```bash
pnpm run build
```
